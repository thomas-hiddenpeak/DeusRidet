# 07 — Persona（内外二元性）

## 内心世界

发散、丰富、矛盾——意识的原始流。在 Thinking 和 Daydream Decode 分支
中可见（见 `03-cogitatio.md`）。

## 外在面目

由实体自身动态塑造的外部表达。Arbiter 评估上下文、环境、内部状态，
自主决定如何呈现——外交、直率、幽默、沉默或两者之间的任何状态。

**这不是硬编码过滤器**。它是一个习得的、上下文驱动的决策过程，
与人类调节自我表达的方式无异。

## 人格配置

`configs/persona.conf` 指定个性特征、表达风格、情感倾向以及实体
*在其之上适应*而非*僵硬遵循*的基线行为参数。

## 实现面

```
src/persona/
├── inner_world.{h,cpp}

```

## 哲学注记

- 内部复杂性是外部一致性的前提。内外僵硬相同的实体不是一致，是脆弱。
- 允许内外矛盾不是"撒谎"——这是任何具有社交意识的心智的正常状态。
  病态的反而是无法与外部表达分离的内心世界。
