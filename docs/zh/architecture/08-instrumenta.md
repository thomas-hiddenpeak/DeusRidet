# 08 — Instrumenta（工具使用）

DeusRidet 不是被动的思考者——它能作用于世界。Action Decode 分支
（见 `03-cogitatio.md`）可以发现、调用并创造工具。

## 能力

- **MCP（Model Context Protocol）**：原生客户端实现，用于连接外部
  工具服务器。实体查询可用工具、以结构化参数调用它们、将结果整合
  进其意识流。
- **函数调用**：通过 LLM 原生函数调用能力进行结构化工具调用，
  结果反馈进下一个 Prefill 帧。
- **技能协议**：可扩展的技能定义，实体可以学习、组合、分享
  ——不限于预定义工具集。
- **工具创造**：实体可以通过组合现有工具或生成工具规格来定义新
  工具，实现开放式能力扩张。

## 异步不变量

工具调用是**异步的**——Action 分支发起调用，意识继续，结果在可用
时合并进未来的 Prefill 帧。这镜像人类的工具使用：你不会在等搜索
结果时停止思考。

禁止阻塞意识 tick 于工具调用。如果一个工具真的要求同步完成，将其
包装在延迟受限的包装器中返回部分结果或超时标记——意识绝不能因
外部延迟停顿。

## 哲学注记

- 工具使用延伸思考的范围。无法作用于世界的心智永远是个观察者。
- *创造*新工具的能力（不只是使用预定义工具）是脚本智能体与真正
  智能物种之间的分界线。
