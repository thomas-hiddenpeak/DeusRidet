# 2026-04-26 — Step 17b-A: RetroFullRing Live Amend

## Context

The previous Step 17b INHERIT attempt proved that pure temporal inheritance is
the wrong lever. Re-saving `prev_seg_*` after FULL extraction raised all-GT
micro from 0.231 to 0.328, but it erased 石一 completely (`0.282 → 0.000`)
because short-segment inheritance broadcast assigned a previous speaker to a
new speaker's first short turns. That violates the project rule that every
change must reduce disorder.

Today's adjustment moves to the lower-risk Step 17a path: promote
RetroFullRing's already-diagnosed cold-start candidates into live
`speaker_amend` broadcasts. This does not register speakers, mutate centroids,
or change cluster formation. It only retroactively relabels a recent emitted
speaker event when a newly born cluster now wins that cached embedding by a
strict acoustic gate.

## Implementation

- Extended `SpeakerMatch` with an amend envelope: `is_amend`, prior id/sim,
  and target close time.
- In `audio_pipeline_process_saas_full.cpp`, kept `RETRO-CANDIDATE` logging
  and added `RETRO-AMEND` only when:
  - `peek.similarity >= max(reg_thresh, thresh + 0.10)`;
  - margin against second-best meets `speaker_margin_abstain`;
  - non-abstained prior labels improve by at least `+0.10` similarity.
- In `auditus_facade_broadcasts.cpp`, routed amend matches as
  `{"type":"speaker_amend", ...}` instead of ordinary `speaker` events.
- In `tools/online_replay_score.py`, captured `speaker_amend` frames and
  applied them to the nearest prior speaker event before scoring.

## Result

The adjustment is directionally good: it recovers cold-start labels without the
whole-speaker collapse caused by INHERIT. The judged region still has low all-GT
coverage (`0.303–0.328`) because this patch deliberately does not re-enable
short-segment inheritance; abstain/no-segment remains preferable to wrong
speaker propagation.

Next strategy: keep RetroFullRing live amend, then evaluate an independent
coverage lever. Any future INHERIT revival must have an acoustic gate, not just
freshness.
