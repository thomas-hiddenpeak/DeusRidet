### A1 lessons

- **Three-commit atomic pattern scales.** Each sub-commit was
  independently compilable and awaken-verified. When A1b hit a stale
  GLOB_RECURSE cache (`undefined reference to process_saas_full_extract_`),
  one `cmake ..` rerun was the fix — no need to unwind A1a.
- **Parameter plumbing is the only delta allowed.** A1a promoted
  `std::vector<int16_t>& pcm_buf` to `const int16_t* pcm_buf` + int; the
  interior `pcm_buf.data()` call had to be updated. Everything else
  was verbatim body transplant.
- **Bash `!` in commit messages triggers history expansion.** Use
  heredoc → `/tmp/msg.txt` + `git commit -F`.
- **cwd discipline.** The awaken ritual must `cd /home/rm01/DeusRidet`
  first; running it from `build/` gives exit 127 because
  `./build/deusridet` is unresolved.
