## Design Crystals

**Why a facade this time (unlike the Step 7d router).** Each of these
three callbacks speaks exactly two subsystems — Conscientia and Nexus —
and every input type (`DecodeResult`, `WakefulnessState`,
`ConscientiMetrics`, speech-token strings) is owned by Conscientia.
That is textbook facade territory: a single cross-boundary seam that
belongs cleanly to one side. Contrast with Step 7d's text-command
router, which fanned across Auditus + Conscientia + Persona and
therefore lived in a peer Actus TU, not a facade.

**Why the JSON helpers went to Communis.** Two facades now emit JSON
envelopes (`auditus_facade`, `conscientia_facade`); any future Memoria /
Persona / Orator facade will need the same `sanitize_utf8` + JSON
`json_escape` pair. Keeping a second copy in `auditus_facade.h` would
have been tech-dynamics — the shortest path. Instead the canonical home
moved to `src/communis/json_util.h`, and `auditus_facade.h` kept
backwards compatibility for every call site by re-exporting through
`using`. Zero call-site churn, one definition, correct layering.
